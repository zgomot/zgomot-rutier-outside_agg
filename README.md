# zgomot-rutier-outside_agg
Hărți de zgomot rutier în afara aglomerărilor din România

Acest proiect oferă o vizualizare web a hărților strategice de zgomot pentru drumurile principale din România, în afara aglomerărilor urbane, realizate conform Directivei 2002/49/CE privind evaluarea și gestionarea zgomotului ambiental.

## Vizualizare hartă

👉 https://zgomot.github.io/zgomot-rutier-outside_agg/

## Tip serviciu

- Serviciu: WMS (Web Map Service)
- Standard: OGC WMS 1.1.1
- Sursă: GeoNoise MMAP

## Descriere

Sunt disponibile următoarele layere:

- **Lden** – nivel de zgomot pentru zi-seară-noapte  
- **Lnight** – nivel de zgomot pentru perioada de noapte  

Aceste layere reprezintă contururi de zgomot (noise contours) generate de traficul rutier pentru drumurile principale situate în afara aglomerărilor urbane.

## Instituție responsabilă

Ministerul Mediului, Apelor și Pădurilor (MMAP)
